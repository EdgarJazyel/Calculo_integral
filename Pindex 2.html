Integral Power

Aprende cálculo integral sin morir en el intento

¿Qué es una integral?

Una integral sirve para encontrar áreas, acumulaciones o recuperar una función original a partir de su derivada.

Regla de la Potencia

Para resolver integrales de potencias de x:

  1. Suma 1 al exponente.
  2. Divide entre el nuevo exponente.
  3. Agrega la constante C.

∫xⁿ dx = xⁿ⁺¹ / (n + 1) + C

Calculadora de Integrales

Escribe una potencia, por ejemplo: x4, 5x2 o 8x6

Ejemplos Resueltos

∫x³dx = x⁴/4 + C

∫5x²dx = 5x³/3 + C

∫7x⁸dx = 7x⁹/9 + C

Caso Especial

Cuando el exponente es -1:

∫1/x dx = ln|x| + C

Aquí NO se aplica la regla de la potencia.

Método de Sustitución

El método de sustitución se usa cuando una función aparece acompañada de su derivada.

Ejemplo Resuelto

∫ (3x²+2)(x³+2x−1)⁵ dx

Paso 1: u = x³ + 2x − 1

Paso 2: du = (3x² + 2)dx

Paso 3: ∫u⁵du

Paso 4: u⁶/6 + C

Paso 5: (x³+2x−1)⁶/6 + C

Integración por Partes

La integración por partes se usa cuando una integral contiene el producto de dos funciones.

∫ u dv = uv − ∫ v du

Para elegir u, se puede usar la regla ILATE:

  • I: Inversa
  • L: Logarítmica
  • A: Algebraica
  • T: Trigonométrica
  • E: Exponencial

Ejemplo Resuelto

∫ x eˣ dx

Paso 1: Elegimos u y dv

u = x

dv = eˣ dx

Paso 2: Derivamos u e integramos dv

du = dx

v = eˣ

Paso 3: Aplicamos la fórmula

∫ u dv = uv − ∫ v du

Paso 4: Sustituimos

∫ x eˣ dx = x eˣ − ∫ eˣ dx

Paso 5: Integramos

x eˣ − eˣ + C

Resultado: x eˣ − eˣ + C
